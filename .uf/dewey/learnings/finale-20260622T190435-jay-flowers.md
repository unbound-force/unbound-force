The /finale command's sub-steps within Step 5 ("Create or Find PR") use lettered sub-steps (a through h). When inserting new sub-steps (like PR template detection or review-council findings), all subsequent sub-step letters must be re-sequenced. The current ordering after the improve-finale-pr-description change is: a=fork detection, b=generate title, c=PR template detection, d=generate body, e=review-council findings, f=show for approval, g=create (with --body-file), h=report URL. Future changes that add new sub-steps to Step 5 must maintain this letter sequence.
